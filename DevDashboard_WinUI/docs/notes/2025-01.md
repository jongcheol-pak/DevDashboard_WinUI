### 핵심 변경 방식

**이전 방식:**
```xml
<!-- XAML -->
<Button Content="{StaticResource Btn_Save}" />
<Button ToolTipService.ToolTip="{StaticResource Dashboard_TooltipPin}" />
```
```csharp
// C#
var text = LocalizationService.Get("key"); // Application.Current.Resources 조회
```

**변경 후 방식:**
```xml
<!-- XAML -->
<Button x:Uid="Btn_Save" />   <!-- Btn_Save.Content = "저장" in resw -->
<Button x:Uid="PinButton" />  <!-- PinButton.[ToolTipService.ToolTip] = "핀 고정/해제" in resw -->
```
```csharp
// C# (API 변경 없음)
var text = LocalizationService.Get("key"); // ResourceLoader.GetString() 조회
```

### 특수 케이스 처리

1. **`<Run>` 요소 (x:Uid 미지원)**: `x:Name="ProjectCountSuffixRun"` + code-behind에서 `LocalizationService.Get` 설정
2. **아이콘 버튼 툴팁**: `x:Uid="IconDeleteBtn"` - resw에 `Content` 없이 `[ToolTipService.ToolTip]`만 정의 (FontIcon Content 덮어쓰기 방지)
3. **동적 Title 다이얼로그** (GroupDialog, ProjectSettingsDialog, HistoryDialog): code-behind에서 Title 설정 유지, x:Uid로 버튼 텍스트만 처리
